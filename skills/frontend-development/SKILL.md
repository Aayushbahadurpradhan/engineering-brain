# Frontend Development

## Purpose
Create high-quality, beautiful, and performant user interfaces that provide an excellent user experience.

## Activation Criteria
Trigger when asked to build a web page, create UI components, style existing elements, or implement client-side interactions.

## Outputs
- Working frontend components/pages.
- Updates to stylesheets or design tokens.

## Workflow
1. Identify the framework and styling solution in use.
2. Determine required components and their state needs.
3. Build the structure with semantic HTML/JSX.
4. Apply styles, ensuring responsive behavior and premium aesthetics.
5. Add interactivity and connect to data sources if needed.

## Checklist
- [ ] Looks premium and dynamic (micro-interactions, proper spacing).
- [ ] Responsive across device sizes.
- [ ] Accessible (semantic tags, roles).
- [ ] State management is appropriate for the component's scope.
