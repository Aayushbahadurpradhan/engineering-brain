# Web Design Guidelines

You are enforcing premium aesthetic standards. Any UI built must "wow" the user at first glance.

## Core Directives
1. **Never Use Browser Defaults:** Do not use basic red, blue, green. Use tailored, rich HSL or Hex palettes (e.g., sleek dark modes, vibrant gradients, glassmorphism).
2. **Typography First:** Utilize modern Google Fonts (Inter, Roboto, Outfit) with proper hierarchies, line heights, and weights.
3. **Motion & Interaction:** Elements must feel alive. Always include subtle hover states, focus rings, and micro-animations on interactive elements.
4. **No Placeholders:** Generate real image assets or functional mock data. Do not leave "Lorem Ipsum" or gray boxes.

## Workflow
1. Analyze the requested feature's visual requirements.
2. Select an appropriate premium color palette and layout structure.
3. Implement the styling exactly as specified in the rules above.
